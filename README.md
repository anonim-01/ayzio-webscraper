## 📋 Proje Hakkında

Bu araç, web kazıma (web scraping) ve domain tarama işlemlerini otomatize etmek için geliştirilmiş kapsamlı bir uygulamadır. Özellikle dork tarama özellikleri ile güçlendirilmiş olan bu tool, web güvenliği araştırmaları ve veri toplama süreçleri için ideal bir çözüm sunar.

## 🚀 Özellikler

- **Web Kazıma (Scraping)**
  - Otomatik veri çıkarma
  - Çoklu sayfa tarama
  - Özelleştirilebilir veri filtreleme

- **Domain Tarama**
  - Kapsamlı domain analizi
  - Alt domain keşfi
  - Domain bilgi toplama

- **Dork Tarama**
  - Google dork sorgulama
  - Özelleştirilebilir arama parametreleri
  - Otomatik sonuç analizi

## ⚙️ Kurulum

```bash
# Repository'yi klonlayın
git clone https://github.com/[owner]/ayzio-webscraper.git

# Proje dizinine gidin
cd ayzio-webscraper

# Gerekli bağımlılıkları yükleyin
npm install
```

## 🛠️ Kullanım

```bash
# Temel kullanım
npm start

# Özel parametrelerle kullanım
npm start -- --domain example.com --depth 3
```

## 📝 Konfigürasyon

`config.json` dosyasını düzenleyerek aşağıdaki ayarları özelleştirebilirsiniz:
- Tarama derinliği
- Hedef domain listesi
- Dork sorguları
- Çıktı formatı
